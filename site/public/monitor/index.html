Glosario

Unidad administrativa: entidad pública a la que fue asignado el presupuesto, por ejemplo ministerios y autónomas.

Programa: división del presupuesto por objetivos o departamentos dentro de la unidad administrativa.

Clasificador: código presupuestario indicando el uso del gasto.

Votado: monto asignado con base en el presupuesto aprobado por la Asamblea Legislativa.

Modificado: presupuesto votado más las modificaciones hechas durante el aǹo.

Variación: suma de las modificaciones realizadas durante el año.

Ejecutado: recursos gastados.

Saldo: remantente del presupuesto modificado que no fue ejecutado.

Eficiencia: indicador del grado en que el presupuesto modificado fue ejecutado.

Credibilidad: indicador de la variación absoluta porcentual entre el presupuesto ejecutado y el votado.

Acceso a la interfase de datos

URL de la API de datos:

La sintaxis para usar la API de datos es la siguiente: "/api/v2/monthly/[AÑO]/[UNIDAD_PRIMARIA]/[PROGRAMA]/[CLASIFICADOR]". Los elementos [PROGRAMA] y [CLASIFICADOR] son opcionales. Si desea obtener los datos para un clasificador, pero sin referirse a un programa en particular, reemplace la opción [PROGRAMA] por la palabra object. Los datos son servidos en formato JSON.

Para acceder a los diccionarios a través de la API de datos, agregue al enlace lo siguiente: "?dict=[ELEMENTO]", donde [ELEMENTO] puede ser: year para los años, office para las unidades administrativas, program para los programas y object para los clasificadores. Por ejemplo, https://fiscal.funde.org/api/v2/monthly/2010/0100?dict=program, devuelve la lista de programas de la Asamblea Legislativa para el año 2010.
